При расчете корня произвольной степени из числа я еще реализовал свою функу возведения в степень. Так, на всякий случай. Чтобы уж метод был полностью ручным. Первый вариант этой программы рассчитан на то, что степень извлечения корня - целое число.
 
Второй вариант проекта - вычисление корня для любого вещественного числа. Тут пришлось отказаться от собственной реализации функции возведения в степень и воспользоваться стандартной библиотечной Math.Pow.
