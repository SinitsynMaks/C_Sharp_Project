В данной программе я реализовал два варианта чтения данных из xml файла:
- через Linq;
- стандартным способом чтения xml файлов.
В папке ReportBuilderClassesIerarhial я сгруппировал иерархические классы для чтения двумя способами.

ClassDiagram - диаграмма классов, поясняющая взаимосвязь классов. 

Наследование основных классов присходит от интерфейсов, чтобы в будущем можно было легко изменить реализация чтения из различных источников данных. 

Класс XmlWriter - некий шаблон для записи в файл. 
Класс DebugWriter - отладочный, для вывода данных на консоль. Классы вывода легко переключить при необходимости. 

Все основные классы разбиты по файлам с соответствующими названиями. 

Уже нет времени написать коментарии. 

Через запросы Linq метод работает в 2 раза быстрее.
